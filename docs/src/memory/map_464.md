# CPC 464 Base Memory Map

This memory map outlines the structural locations of the firmware operating system, workspaces, and system stacks for the base Amstrad CPC 464 model.

| Address Range | Size | Allocation / Description |
| :--- | :--- | :--- |
| `&0000 - &003F` | 64 Bytes | **Restart Block:** System vectors, restart pointers (`RST 0` to `RST 7`), interrupt entries, and the low jump block. |
| `&0040 - &016F` | 304 Bytes | **BASIC Input Area:** Contains the tokenized command buffer for direct user input. |
| `&0170 - &A6FB` | ~41 KB | **BASIC Working Space:** Program source code storage, variable tables, arrays, and general free user RAM. |
| `&A6FC - &A6FF` | 4 Bytes | **AMSDOS Chaining Block:** Pointer addresses for disk ROM chaining. |
| `&A700 - &ABAF` | 1200 Bytes| **AMSDOS Reserved Buffer:** General disk work area (moved down if ROM expansion numbers exceed 8 on expanded hardware). |
| `&ABB0 - &ABFF` | 80 Bytes | Unused / Reserved space. |
| `&AC00 - &B06E` | ~1.1 KB | **System Workspace:** Contains active event blocks, timer tickers, keyboard buffer arrays, sound envelopes, and line drawing buffers. |
| `&B06F - &B284` | 534 Bytes | **Firmware Stacks:** General workspace for GOSUB, FOR/NEXT loops, and variable name registers. |
| `&B285 - &B7C2` | ~1.3 KB | **Stream Parameter Blocks:** Attributes for screen streams 0–7 (cursor coordinates, active pens, and window sizes). |
| `&B7C3 - &B833` | 113 Bytes | **Display System Variables:** Mode state variables, screen baseline, active inks, and first/second flash durations. |
| `&B834 - &B8B3` | 128 Bytes | **Interrupt Temporary Stack:** Used by the firmware during interrupt context switching. |
| `&B8B4 - &B8FA` | 71 Bytes | **Kernel Timers & ROM Info:** Master time counter, ticker chains, and active upper ROM tracking tables. |
| `&B900 - &B923` | 36 Bytes | **High Kernel Jumpblock:** Entry points to ROM paging routines. |
| `&B924 - &BAE3` | 448 Bytes | High Kernel workspace and internal firmware utilities. |
| `&BB00 - &BDFF` | 768 Bytes | **Firmware Redirect Jumpblocks:** Landing vectors for all sub-packs (Key, Text, Graphics, Screen, Cassette, Sound, Kernel, and Maths). |
| `&BE00 - &BE7F` | 128 Bytes | **Disk Drive Calibration Block:** Motor timelines, step rates, and seek timing arrays. |
| `&BE80 - &BFFF` | 384 Bytes | **Machine Stack:** Space for stack pointer (`SP`) operations (limited to `&BFFF` upper boundary). |
| `&C000 - &FFFF` | 16 KB | **Video Memory Block:** Contains active screen raster data. Map offsets are adjusted by hardware scrolling commands. |
